###  Instruction to create python 3.14.4 enviroment for running this project.<br>
First, create the python virtual enviorment <br>
--> "py -3.14 -m venv salem1692_python_venv" <br>
--> "conda create --name salem1692_python_venv python=3.14.4" <br>
<br>
Second, activate that virtual evironment<br>
--> "cd salem1692_python_venv/Scripts" <br>
--> "activate.bat" <br>
<br>
--> "conda list" <br>
--> "conda activate salem1692_python_venv" <br>
<br>
 Then install our "salem1692_requirements.txt" to your local virtual environment<br>
-->  "pip install -r salem1692_requirements.txt"<br>
--> "conda install --file salem1692_requirements.txt" <br>
<br>
 Ready to run the main file "python launcher.py" <br>
